# Contributing to AD-STG

Thank you for your interest in contributing to the Active Directory Security Testing Guide!

## How to Contribute

### Reporting Issues

- Use GitHub Issues for bug reports and feature requests
- Search existing issues before creating a new one
- Provide detailed information

### Submitting New Techniques

We welcome submissions of new attack techniques! Please include:

1. **Technique Description**
   - Clear objective
   - Detailed test methodology
   - Tool recommendations

2. **Remediation Guidance**
   - Preventive controls
   - Detective controls

### Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Commit with clear messages
5. Create a Pull Request

## Questions?

- Check existing documentation
- Search GitHub Issues
- Ask in GitHub Discussions
