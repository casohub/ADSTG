# Getting Started with AD-STG

This guide will help you get started with the Active Directory Security Testing Guide.

## Prerequisites

- Understanding of Active Directory fundamentals
- Familiarity with penetration testing concepts
- Access to testing tools

## Installation

1. Clone this repository
2. Review the methodology structure
3. Install recommended tools

## Your First Assessment

1. Start with [AD-00 Pre-Engagement](../methodology/AD-00-Pre-Engagement/)
2. Follow controls sequentially
3. Document findings using templates
